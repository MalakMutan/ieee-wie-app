# IEEE WIE App 👩‍🔬📱

A simple mobile app built with React Native for the IEEE Women in Engineering (WIE) chapter at Al-Quds University.

🎯 Project Overview

This app provides a welcoming interface for users with two main options:
- Register as an IEEE Member** – allows users to enter their name and IEEE ID and see the list of registered members.
- Continue as a Visitor** – redirects to the IEEE WIE Instagram page.

---

 🧩 Main Screens

- Welcome Screen**: Displays the WIE logo and two buttons (Member | Visitor).
- Member Screen**: Allows users to enter their name and IEEE ID and displays all registered members.
- Visitor Screen**: Opens the IEEE WIE Instagram page in the browser.

---

 ⚙️ Requirements

- Node.js
- Expo CLI
- Git
- Android/iOS device or emulator

---

 🚀 How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/MalakMutan/ieee-wie-app.git
   cd ieee-wie-app
````

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the Expo development server:

   ```bash
   npx expo start
   ```

4. **Run the app** on your phone using the Expo Go app or on an emulator.
